Création d'une machine virtuelle

1. Téléchargement des outils

Nous avons téléchargé et installé VirtualBox. C'est un hyperviseur gratuit, open source et largement utilisé pour la virtualisation.

Ensuite, nous avons récupéré l’image ISO d’Ubuntu. Ubuntu est une distribution Linux populaire, réputée pour sa stabilité.

2. Création d'une nouvelle machine virtuelle

Dans VirtualBox, nous avons nommé la machine virtuelle "MonVM" et défini le répertoire où elle serait stockée. La sélectionne de l'image ISO d'Ubuntu a permis de configurer automatiquement les paramètres de type, sous-type et version.

Nous avons laissé les paramètres par défaut pour l'option "Installation sans intervention" car nous voulions effectuer une installation standard.

3. Configuration des ressources matérielles

Nous avons alloué 4 Go de RAM (12.5% de la machine hôte) et 2 sur les 24 cœurs de processeur disponibles à la machine virtuelle.

Nous avons attribué 20 Go d'espace de stockage à la machine virtuelle en choisissant le type de fichier disque dur VDI (Virtual Disk Image), Le format natif pour VirtualBox.

4. Démarrage et installation d'Ubuntu

Après avoir démarré la machine virtuelle, nous avons suivi le processus d’installation d'Ubuntu. Cela inclut la sélection de la langue et la configuration de l’utilisateur.

Ubuntu a été installé sur notre MV après quelques minutes de chargement. .


Création d’un compte administrateur

1. Ouverture du terminal et passage en mode root

Nous avons d'abord ouvert le terminal sous Ubuntu.

Nous sommes passés en mode superutilisateur (root) en utilisant la commande sudo -i pour ne pas avoir à répéter sudo, puis avons mis à jour les paquets en utilisant apt update && apt upgrade.

2. Ajout de l’utilisateur au groupe "sudo"

Nous avons utilisé la commande adduser admin_Gr10_Binome04 pour créer un compte individuel avant de lui attribuer des privilèges administratifs.

Pour donner des privilèges administratifs au nouvel utilisateur, nous avons utilisé la commande usermod -aG sudo admin_Gr10_Binome04 Cela ajoute l’utilisateur au groupe "sudo". Nous avons testé si le nouvel utilisateur dispose bien des droits administratifs en exécutant la commande cat /etc/group | grep sudo


Création d’un groupe

1. Création de deux utilisateurs et un groupe

Nous avons utilisé la commande adduser Binome04_1 et adduser Binome04_2 pour créer deux nouveaux utilisateurs.

Pour créer un nouveau groupe, nous avons utilisé la commande: groupadd Binome04. Cette commande permet de créer un groupe avec le nom spécifié.

2. Ajout au groupe et vérification

Pour ajouter les utilisateurs existants au nouveau groupe, on a utilisé :
usermod -aG Binome04 Binome04_1 et usermod -aG Binome04 Binome04_2

Nous avons vérifié si l’utilisateur a bien été ajouté au groupe avec :
groups Binome04_1 et groups Binome04_2


Personnalisation d’une invite de commande (prompt)

1. Affichage du nom dans le prompt

Modifions le fichier ~/.bashrc avec : nano ~/.bashrc.
Ajoutons à la fin : PS1='\u@\h:\w\$'.

Rechargeons avec : source ~/.bashrc.
Le prompt affiche l’utilisateur, l’hôte et le répertoire courant.


Installation et configuration d'un service web

1. Configuration réseau

Changeons le mode réseau de la machine virtuelle de NAT à Accès par Pont dans les paramètres réseau de VirtualBox.

Utilisons la commande ip addr show pour obtenir l’adresse IP de la machine virtuelle. Notez l’adresse affichée sous l’interface active.

2. Configuration SSH

Installons le serveur OpenSSH avec : sudo apt install openssh-server.

Configurons SSH en modifiant le fichier /etc/ssh/sshd_config, puis redémarrons le service avec : sudo systemctl restart ssh.

3. Transfert des fichiers en utilisant SSH

Transférons notre dossier site web vers la machine virtuelle.

Vérifions que le dossier est bien transféré avec la commande ls sur la machine.

4. Installation et configuration Apache2

Installons Apache2 avec : sudo apt install apache2.

Modifions les fichiers de configuration, puis redémarrons Apache avec : sudo systemctl reload apache2.

5. Résultat

Accédons au site web hébergé sur la machine en saisissant son adresse IP dans un navigateur.
